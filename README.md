# 🎯 Horizontal Throw

Horizontal Throw is a simple **horizontal projectile motion simulator** built with **HTML, CSS, JavaScript, Bootstrap, and Highcharts**.  
It lets you enter an initial speed and two launch angles to compare projectile paths visually.

---

## 🚀 Features
- Input speed (m/s) and launch angles (°).
- Interactive chart showing two trajectories.
- Responsive design with Bootstrap.
- Educational physics visualization.

---

## 📐 Physics Model
- Horizontal: `x = v * cos(θ) * t`  
- Vertical: `y = v * sin(θ) * t - 0.5 * g * t²`  
- Gravity `g ≈ 10 m/s²`
